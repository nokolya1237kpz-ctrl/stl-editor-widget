## ✨ Возможности

### ✂️ Универсальный редактор STL
| Функция | Описание |
|---------|----------|
| 📝 Текст | Вырезай или добавляй выпуклый текст (до 50 символов, кириллица/латиница) |
| 🕳️ Отверстия | Цилиндрические сквозные или глухие отверстия с настройкой радиуса и глубины |
| 👁️ Глаза | Два сферических отверстия для игрушек, масок, фигурок |
| ✂️ Вырез кубом | Прямоугольный вырез из центра модели для пазов и модификаций |
| 🖼️ Превью | Мгновенное 2D-превью результата перед скачиванием |

### 🔑 Генератор брелков (ПРО)
| Параметр | Диапазон | Описание |
|----------|----------|----------|
| 📐 Форма | `📦 Прямоугольник` / `🔵 Овал` / `❤️ Сердце` | Выбор геометрии основы |
| 📏 Размеры | Ширина 30–80 мм, Высота 20–60 мм, Толщина 2–6 мм | Гибкая настройка габаритов |
| 🔤 Текст | До 50 символов, размер 4–16 мм | Надпись на брелке |
| 🔻🔺 Режим текста | Вырезанный / Выпуклый | Углубление или рельеф |
| 🔗 Петелька | Авто-генерация | Отверстие для кольца + крепление для шнурка |

### 🧱 CAD-сборка (Tinkercad-style)
| Примитив | Параметры | Особенности |
|----------|-----------|-------------|
| 📦 Куб | Ширина × Высота × Глубина (1–100 мм) | Базовый строительный блок |
| 🔵 Цилиндр | Радиус (1–50 мм), Высота (1–100 мм) | Круглые элементы, трубы |
| ⚪ Сфера | Радиус (1–50 мм) | Шары, кнопки, декор |
| 🔺 Конус | Радиус основания, радиус вершины (0 = пирамида), Высота | Воронки, наконечники, острые формы |

**Дополнительные возможности CAD:**
- 🎯 Позиционирование по координатам X/Y/Z
- 🔄 Поворот по осям (градусы)
- ➕ Объединить / ➖ Вычесть — булевы операции для каждой фигуры
- 🗑️ Удаление объектов по одному

### 📦 3D-просмотр
- 🔄 Вращение, масштаб, панорамирование мышкой или касанием
- 🌓 Авто-определение тёмной/светлой темы системы
- 📱 Полностью адаптивный дизайн (мобильные, планшеты, десктоп)
- 📸 Кнопка «Скачать скриншот» для превью

## 🚀 Быстрый старт

### Вариант А: Использовать онлайн (рекомендуется)
1. Открой [Mini App во ВКонтакте](https://vk.com/app54580452)
2. Выбери вкладку: «✂️ Редактор», «🔑 Брелок» или «🧱 CAD»
3. Следуй подсказкам на экране
4. Скачай готовый STL и отправляй на печать!

# STL Мастер — редактор 3D-моделей для ВКонтакте

![Version](https://img.shields.io/badge/version-1.2.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Status](https://img.shields.io/badge/status-active-28a745)

STL Мастер помогает быстро подготовить модель к 3D-печати прямо в браузере: загрузить STL, добавить текст, сделать отверстия, собрать простую CAD-модель, сгенерировать брелок или превратить простой SVG в STL.

## Возможности

### Редактор STL

| Функция | Описание |
|---|---|
| Текст | Добавление выпуклого текста или попытка вырезать текст из модели, до 50 символов |
| Отверстия | Сквозные или глухие цилиндрические отверстия с радиусом, глубиной и смещением |
| Глаза | Два сферических выреза для игрушек, масок и фигурок |
| Вырез кубом | Прямоугольный вырез из модели |
| SVG в STL | Конвертация простых SVG-фигур в вытянутую STL-модель |
| 2D-превью | PNG-превью результата от сервера |

Поддержка SVG рассчитана на простые элементы: `rect`, `circle`, `ellipse`, `polygon`, `polyline` и простые `path` с командами `M/L/H/V/Z`. Сложные кривые SVG лучше предварительно упростить в векторном редакторе.

### Инструменты для печати

- Анализ STL и базовая оценка пригодности к печати.
- Восстановление сетки.
- Авто-ориентация модели.
- Автоисправление модели.
- ZIP-отчёт с моделью, JSON и превью.
- Помощник слайсера для PLA, PETG, ABS, TPU и RESIN.
- Режим новичка, каркасное превью и прозрачный фон PNG.

### Генератор брелков

| Параметр | Возможности |
|---|---|
| Шаблоны | Классический, компактный, овальный бейдж, длинная бирка, сердце |
| Формы | Прямоугольник, овал, сердце |
| Размеры | Ширина 30-80 мм, высота 20-60 мм, толщина 2-6 мм |
| Текст | До 50 символов, размер 4-16 мм, русский/английский/авто |
| Режим текста | Выпуклый или вырезанный |
| Отверстие | Настраиваемый диаметр, для сердца можно менять позицию |

### CAD-конструктор

| Примитив | Параметры |
|---|---|
| Куб | Ширина, высота, глубина |
| Скруглённый куб | Размеры и радиус скругления |
| Цилиндр | Радиус и высота |
| Трубка | Внешний радиус, внутренний радиус, высота |
| Сфера | Радиус |
| Конус | Радиус основания, радиус вершины, высота |
| 3D-текст | Текст, размер, толщина |

Дополнительно доступны позиция X/Y/Z, поворот по осям, масштаб X/Y/Z, операции объединения и вычитания. Есть готовые CAD-шаблоны: подставка, трубка, табличка.

### 3D-просмотр

- Вращение, приближение и перемещение модели через Three.js.
- Автоцентрирование и постановка модели на стол.
- Ориентация выбранной стороной вниз.
- Экспорт текущей модели в STL после клиентских поворотов.

## Быстрый старт

### Онлайн

1. Открой [Mini App во ВКонтакте](https://vk.com/app54580452).
2. Выбери вкладку: `Редактор`, `Брелок` или `CAD`.
3. Загрузи STL, SVG или настрой параметры модели.
4. Проверь 3D-просмотр и скачай готовый STL.

### Локально для фронтенда

```bash
cd "Фронт энд 3д мастер"
python3 -m http.server 8000
```

Открой в браузере: `http://localhost:8000`.

Фронтенд по умолчанию обращается к API:

```js
const API_BASE = 'https://3dcalk.freedynamicdns.net:8443/api';
```

Для локальной разработки backend можно заменить `API_BASE` в `script.js` на адрес своего Flask-сервера.

## Backend API

Основные endpoints:

| Endpoint | Назначение |
|---|---|
| `POST /api/stl/editor-advanced` | Текст, отверстия, глаза, вырез кубом для STL |
| `POST /api/convert/svg-to-stl` | Конвертация простого SVG в STL |
| `POST /api/generate/keychain-advanced` | Генерация брелков |
| `POST /api/cad-editor/build` | Сборка CAD-модели из примитивов |
| `POST /api/stl/analyze` | Анализ STL |
| `POST /api/stl/repair` | Восстановление STL |
| `POST /api/stl/auto-orient` | Авто-ориентация |
| `POST /api/stl/auto-fix` | Автоисправление |
| `POST /api/stl/print-advisor` | Советы для слайсера |
| `GET /api/stl/preview/<id>.png` | PNG-превью результата |

Ответы со STL могут содержать заголовки `X-Preview-Id`, `X-Preview-Path`, `X-Download-Url` и `X-Download-Filename`.

## Ограничения и безопасность

- Максимальный STL на клиенте и сервере: 50 МБ.
- Сервер ограничивает слишком плотные сетки по количеству вершин и треугольников.
- Вырезанный текст может быть тяжёлой операцией для сложных STL. Если операция не проходит, используйте выпуклый режим или упростите модель.
- Для кириллицы на сервере нужен TTF-шрифт с поддержкой русского текста: DejaVu Sans, Noto Sans или Roboto.
- SVG-конвертер не является полноценным CAD-импортом SVG: сложные кривые и составные path лучше упростить заранее.

## Проверка перед публикацией

```bash
python3 -m py_compile app.py
node --check script.js
```

После загрузки на сервер проверь вручную:

- добавление выпуклого текста на STL;
- вырезанный текст на простой модели;
- генерацию брелка по каждому шаблону;
- SVG в STL на простом прямоугольнике или логотипе без кривых;
- CAD-шаблоны и сборку модели;
- загрузку 2D-превью после операций.

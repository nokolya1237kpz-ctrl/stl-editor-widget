# ✂️ STL Мастер — Редактор 3D-моделей для ВКонтакте

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

> 🎨 Редактируй, генерируй и собирай 3D-модели прямо в браузере!

## 🔗 Демо

👉 [Открыть Mini App во ВКонтакте](https://vk.com/app<APP_ID>)

## ✨ Возможности

### ✂️ Редактор STL
- 📝 Добавляй текст (вырезанный или выпуклый)
- 🕳️ Вырезай цилиндрические отверстия
- 👁️ Добавляй глаза-сферы для игрушек
- ✂️ Вырезай кубом из центра модели
- 🖼️ PNG-превью перед скачиванием

### 🔑 Генератор брелков
- 🎨 Текст с выбором режима (cut/emboss)
- 📏 Гибкие размеры основы
- 🔗 Автоматическая петелька для кольца

### 🧱 CAD-сборка (Tinkercad-style)
- 📦 Кубы и 🔵 цилиндры как примитивы
- 🎯 Позиционирование по X/Y/Z
- 🔗 Объединение объектов в одну модель

### 📦 3D-просмотр
- 🔄 Вращение, масштаб, панорамирование
- 🎨 Тёмная/светлая тема
- 📱 Адаптивный дизайн для мобильных
